CONTROLS:
	← - MOVE LEFT
	↑ - MOVE UP
        → - MOVE RIGHT
	↓ - MOVE DOWN

TROUBLESHOOTING:

- the game doesn't start:
	-- ensure you have Python and Pygame installed. (if not, download python and run "pip install pygame" on your command prompt to install pygame.)

- assets missing:
	-- make sure youve extracted the files properly and the assets are in the same dir as the main.py file.

- any other errors:
